How to Master a Song for Spotify and Apple Music

Reading time: 22 min Category: Tutorials

Understanding Streaming Platform Requirements

Mastering for streaming platforms requires a different approach than traditional mastering. At Sweet Dreams Studio, we've optimized thousands of tracks for streaming, and we'll show you how to achieve the best possible sound on every platform.

Platform-Specific Requirements

Spotify:

  • Loudness Target
    • -14 LUFS integrated
    • -1 dB true peak maximum
    • -2 dB true peak recommended
  • Dynamic Range
    • 8-11 LU recommended
    • Varies by genre
    • Affects playlist placement

Apple Music:

  • Loudness Target
    • -16 LUFS integrated
    • -1 dB true peak maximum
    • Sound Check enabled by default
  • Dynamic Range
    • 8+ LU recommended
    • More dynamic range preferred
    • Better quality preservation

Sweet Dreams Studio Tip:

Master to -14 LUFS for the best compromise across platforms. This prevents excessive processing while maintaining competitive loudness.

Essential Mastering Tools

Required Software:

  • Loudness Meters
    • LUFS measurement
    • True peak detection
    • Dynamic range analysis
  • Limiters
    • True peak limiting
    • Clean limiting algorithm
    • Oversampling capability
  • Analysis Tools
    • Spectrum analyzer
    • Phase correlation
    • Stereo imaging

The Streaming Mastering Process

Step-by-Step Approach:

  • 1. Reference Level Setting
    • Import reference tracks
    • Match perceived loudness
    • Compare frequency balance
  • 2. Tonal Balance
    • Corrective EQ
    • Dynamic EQ if needed
    • Multiband compression
  • 3. Dynamic Control
    • Set target dynamic range
    • Apply compression carefully
    • Preserve musical dynamics
  • 4. Final Limiting
    • Target streaming LUFS
    • Control true peaks
    • Maintain transients

Advanced Streaming Optimization

1. Codec Preview:

  • Benefits:
    • Hear codec artifacts
    • Adjust processing accordingly
    • Prevent streaming issues
  • Tools:
    • Codec preview plugins
    • Reference encoders
    • A/B comparison

2. Bass Management:

  • Techniques:
    • Control sub frequencies
    • Monitor mono compatibility
    • Ensure translation
  • Tools:
    • Multiband processing
    • Bass enhancement
    • Phase alignment

Professional Insight:

Use true peak limiting with at least 4x oversampling to prevent inter-sample peaks that can cause streaming artifacts.

Genre-Specific Considerations

EDM/Pop:

  • Characteristics:
    • Higher perceived loudness
    • Controlled dynamics
    • Punchy transients
  • Approach:
    • More aggressive limiting
    • Careful bass control
    • Enhanced clarity

Rock/Metal:

  • Characteristics:
    • Dense midrange
    • Guitar focus
    • Impact preservation
  • Approach:
    • Multiband compression
    • Mid-side processing
    • Dynamic EQ

Jazz/Classical:

  • Characteristics:
    • Wide dynamics
    • Natural sound
    • Minimal processing
  • Approach:
    • Gentle limiting
    • Preserve dynamics
    • Clean EQ

Common Streaming Issues

1. Codec Artifacts:

  • Causes:
    • Over-processing
    • Excessive limiting
    • Complex high frequencies
  • Solutions:
    • Gentler processing
    • High-frequency control
    • Preview codecs

2. Level Inconsistency:

  • Causes:
    • Incorrect LUFS target
    • Dynamic range issues
    • Poor metering
  • Solutions:
    • Use proper meters
    • Check multiple platforms
    • Monitor reference tracks

3. Bass Problems:

  • Causes:
    • Codec compression
    • Phase issues
    • Excessive sub bass
  • Solutions:
    • Control low end
    • Check mono compatibility
    • Use reference tracks

Quality Control Checklist

  • Loudness Targets
    • Check integrated LUFS
    • Verify true peaks
    • Monitor dynamic range
  • Codec Testing
    • Preview compressed audio
    • Check multiple platforms
    • Compare to references
  • System Checks
    • Listen on multiple devices
    • Check in mono
    • Verify translation

Sweet Dreams Studio Secret:

Always check your masters on actual streaming platforms before final release. Sometimes what sounds perfect in the DAW needs adjustment for streaming.

Final Export Settings

  • File Format
    • WAV or AIFF
    • 16-bit/44.1kHz minimum
    • 24-bit recommended
  • Dither
    • Apply if reducing bit depth
    • Use noise shaping
    • Choose appropriate type
  • Metadata
    • Include song information
    • Add ISRC if available
    • Set appropriate tags
